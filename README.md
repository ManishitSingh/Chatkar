## Run Locally

Clone the project

```bash
  git clone https://github.com/MoneyAsk/Chatkar.git
```

Go to the frontend directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

To start server

```bash
  cd backend
  npm run start
```
